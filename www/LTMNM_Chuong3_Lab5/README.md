# LAB 5 - Tương tác cơ sở dữ liệu với MySQL và PDO

Project hoàn thành đủ 12 bài tập trong tài liệu Lab 5. Mỗi bài nằm trong thư mục `exercises` và hiển thị kết quả bằng HTML thuần, không sử dụng CSS.

## Cách chạy với XAMPP

1. Chép thư mục `Lab5` vào `htdocs`.
2. Mở phpMyAdmin, chọn **Import**, rồi nhập file `database/lab3_shop.sql`.
3. Nếu tài khoản MySQL khác mặc định, sửa `config/database.php`.
4. Mở `http://localhost/Lab5/`.

## Cấu hình mặc định

- Host: `localhost`
- Port: `3306`
- Database: `lab3_shop`
- Username: `root`
- Password: rỗng

Lưu ý: Bài 03 không có dòng kết quả với dữ liệu mẫu vì không loại hàng nào có hơn 5 sản phẩm. Đây là kết quả đúng theo yêu cầu đề bài.
